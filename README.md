# AquiFest

Aplicativo mobile para mostrar os festivais da região do Tapajós.

## Funcionalidades

- Página inicial com destaque para festivais ativos
- Cadastro e login de usuários
- Login de administrador e usuários comuns
- Administradores podem criar, editar, excluir, suspender e reativar festivais
- Usuários podem visualizar festivais e confirmar presença
- Festivais suspensos saem da Home e continuam visíveis na lista com indicação de status

## Como executar

1. Instale as dependências: `npm install`
2. Inicie o projeto com `npx expo start`
3. Abra no Expo Go, emulador Android ou navegador web

Para abrir no Android use o emulador do Android Studio ou `npx expo run:android` quando um dispositivo estiver conectado.
