# AquiFest

Aplicativo mobile para mostrar os festivais da região do Tapajós.

## Funcionalidades

- Página inicial com opções de login, cadastro de usuário e cadastro de festival
- Cadastro de usuário com nome, idade, sexo, email e cidade
- Login de administrador e usuários
- Administradores podem cadastrar festivais com nome, imagem do folder, localização, história e período
- Usuários podem visualizar festivais e confirmar presença

## Como executar

1. Instale as dependências: `npm install`
2. Execute no navegador: `npm run web`
3. Abra o app no browser web

Para abrir no Android use o emulador do Android Studio ou `npx expo run:android` quando um dispositivo estiver conectado.
